# Pashword Generator

A Bash script for fast, and secure password generation.
There isn't a typo in the script name. It's intentional. :)

## Features

- Randomly generates password using OpenSSL
- Allows you to select the length and complexity of the password
- Can generate a password using lowercase and uppercase letters, numbers, symbols, as well as hexadecimal passwords

## Requirements

- Linux-based OS
- OpenSSL installed

## Usage

```bash
chmod +x pashword-generator.sh
./pashword-generator.sh
```
